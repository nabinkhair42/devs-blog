# Developer Blog Template

Lightweight blogging platform built with Next.js, MDX, and Tailwind CSS. Perfect for developers.


# Get Started:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/nabinkhair42/devs-blog.git
    cd devs-blog
    ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run the Development Server**:
   ```bash
   npm run dev
   ```
4. **Open in Browser**:
   Navigate to `http://localhost:3000` to see your blog in action.

5. **Create New Blog Posts**:
   Add new MDX files in the `content/blogs` directory.

6. **Customize Your Blog**:
   - Update site details in `src/lib/details.ts`.

# Features

- **MDX Support**: Write blog posts using Markdown and embed React components.
- **Tailwind CSS**: Easily style your blog with utility-first CSS.
- **SEO Optimized**: Built-in SEO features for better search engine ranking.
- **Static Site Generation**: Fast load times with pre-rendered pages.
